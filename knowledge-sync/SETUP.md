# 飞书应用配置指南

## ⚠️ 当前问题

API 返回 400/404 错误，通常是因为：
1. 应用未发布
2. 权限未正确配置
3. 应用未添加到工作区

---

## ✅ 完整配置步骤

### 步骤 1：登录飞书开放平台

访问：https://open.feishu.cn/

### 步骤 2：创建或选择应用

1. 点击左侧 **应用开发** → **企业内部开发**
2. 点击 **创建应用** 或选择已有应用
3. 应用名称：例如 "OpenClaw 知识库"

### 步骤 3：配置应用凭证

1. 进入应用管理页
2. 点击 **凭证与基础信息**
3. 复制：
   - **App ID**: `cli_a924dc7790625bd3` ✓ (已有)
   - **App Secret**: 点击"查看"获取 ✓ (已有)

### 步骤 4：添加权限 ⚠️ 关键步骤

1. 点击左侧 **权限管理**
2. 点击 **申请权限** 或 **批量导入**
3. 搜索并添加以下权限：

```
bitable:app          - 读取多维表格应用信息
bitable:table        - 读取数据表
bitable:record       - 读取记录（如果需要读取数据）
drive:file           - 访问云文档（可选）
```

4. **保存** 权限配置

### 步骤 5：发布应用 ⚠️ 必须步骤

1. 点击左侧 **版本管理与发布**
2. 点击 **创建版本**
3. 填写版本号（如 1.0.0）和更新说明
4. 点击 **提交审核**
5. 企业内部应用通常 **自动审核通过**
6. 等待状态变为 **已发布**

### 步骤 6：将应用添加到工作区

1. 点击左侧 **应用信息**
2. 在 **可见范围** 中添加：
   - 整个组织，或
   - 特定部门/人员
3. 确保你能访问这个应用

### 步骤 7：授权多维表格

1. 打开你的多维表格：https://taifenxi.feishu.cn/base/BFQjbYgHUaaAmwsYdubcZ5hWnc5
2. 点击右上角 **...更多** → **应用设置** 或 **协作**
3. 添加你的应用为协作者
4. 或者在应用后台配置可访问的表格

## 📋 常见错误

### "400 Bad Request"
- 权限未配置
- 应用未发布

### "404 Not Found"
- App Token 或 Table ID 错误
- 应用没有访问该表格的权限

### "403 Forbidden"
- Token 过期
- 权限不足

### "99991661" (app not visible)
- 应用未添加到工作区
- 可见范围未配置

## 💡 快速解决方案

如果以上步骤太复杂，可以试试：

### 方法 1：使用飞书内置机器人

1. 在多维表格中点击 **自动化**
2. 创建 **机器人**
3. 配置 Webhook
4. 用 Webhook 方式推送数据

### 方法 2：导出为 CSV/Excel

1. 在多维表格中点击 **...更多**
2. 选择 **下载为** → **Excel/CSV**
3. 手动放到知识库目录

### 方法 3：使用飞书集成平台

1. 访问 https://open.feishu.cn/platform
2. 使用 **集成平台** 创建数据同步流程
