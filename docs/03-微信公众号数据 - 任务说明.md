## 🎯 功能描述

自动抓取微信公众号文章数据，同步到 GrowingIO 分析平台，生成阅读/分享报表。

## ✨ 核心功能

### 1. 多医院支持

**配置示例** (`hospitals.yml`):
```yaml
hospitals:
  - name: 杭州师范大学附属医院
    wechat_id: "hzsfyy"
    enabled: true
    
  - name: 浙江省中医院
    wechat_id: "zjszyy"
    enabled: true
    
  - name: 上海市中医医院
    wechat_id: "shszy"
    enabled: true
```

### 2. 数据抓取

**抓取内容**:
- 文章标题
- 发布时间
- 阅读数
- 分享数
- 在看数
- 点赞数

### 3. GrowingIO 同步

**接口**:
- `track_custom_event` - 上报自定义事件

**配置**:
```python
CONFIG = {
    "growingio_project_id": "8931a76e733f77f9",
    "growingio_user_id": "hospital_wechat",
    "server_host": "https://ngari-collect.fenxiti.com/",
}
```

### 4. 报表生成

**输出**:
- 每日阅读/分享趋势图
- 医院排名
- 热门文章 Top10

## 🔗 GrowingIO 集成

### 事件上报

```python
def track_custom_event(event_name, attributes):
    """
    上报自定义事件到 GrowingIO
    
    Args:
        event_name: 事件名称（如 "OA_officialNewsDailyData3"）
        attributes: 事件属性字典
    """
    payload = {
        "event_time": int(time.time() * 1000),
        "event_name": event_name,
        "attributes": attributes
    }
    # 发送到 GrowingIO
    requests.post(url, json=payload)
```

### 事件名称规范

- `OA_officialNewsDailyData3` - 公众号每日数据
- `OA_readUser_di` - 阅读用户
- `OA_shareUser_di` - 分享用户

### 属性格式

```python
attributes = {
    "oa_readUser_cnt": 1234,      # 阅读人数
    "oa_readTimes_cnt": 5678,     # 阅读次数
    "oa_shareUser_cnt": 234,      # 分享人数
    "oa_readDeliveryRate": 0.19,  # 阅读完成率
    "oa_readFinishRate": 0.85,    # 阅读完成率
    "event_time": 1772208000000   # 事件时间戳
}
```
