## ✨ 核心功能

### 1. 🔍 自动单位检测

**问题**: Excel 中数据单位可能是"元"或"万元"，之前版本经常搞错导致数据异常。

**解决**:
```python
def detect_unit(df, column_name, sample_row=4):
    sample_value = df.iloc[sample_row][column_name]
    if sample_value > 10000:
        return '元'      # 如 230991.46 → 需要除以 10000
    elif sample_value < 100:
        return '万元'    # 如 23.099146 → 无需转换
    else:
        return '元'      # 边界情况，默认按元处理
```

### 2. ✅ 数据验证

**验证规则**:
- 日流水范围：1000 元 ~ 10 亿元
- 环比范围：-1000% ~ +10000%
- 分项之和验证：与总计差异 < 1%

**示例输出**:
```
⚠️  数据验证警告:
   - 2026-01-06: 环比 -210.69% < 最小值 -1000%
   - 2026-04-08: 日流水 0.0000 万 < 最小值 0.1 万
```

### 3. 🔄 自动去重更新

**逻辑**:
```python
# 1. 获取已有日期
existing_dates = set(...)

# 2. 删除将要更新的记录
DELETE FROM table WHERE date IN (dates_to_update)

# 3. 插入所有记录
INSERT OR REPLACE INTO table VALUES (...)
```

### 4. 🗑️ 删除未来空数据

自动删除日期 >= 今天 且 流水=0 的记录。

## 📊 数据单位说明

### Excel 中的单位

| 列名 | 单位 | 示例值 |
|------|------|--------|
| 咨询流水、处方流水等 | **元** | 15797.8 |
| 日总流水 | **元** | 230991.46 |
| 日流水增量 | **元** | 24354.77 |
| 日流水环比 | **比率** | 0.1178 (= 11.78%) |

### 数据库中的单位

| 字段名 | 单位 | 存储值 |
|--------|------|--------|
| ywul_class_flow, chufang_flow 等 | **万元** | 1.57978 |
| daily_total_flow | **万元** | 23.099146 |
| daily_flow_increment | **万元** | 2.435477 |
| daily_flow_ratio | **比率** | 0.1178 |

### 转换公式

```python
# 所有流水列：元 → 万元
万元 = 元 / 10000

# 环比列：保持不变
比率 = 比率
```

如果看到大量环比警告，可能是阈值设置过严：

```python

```
